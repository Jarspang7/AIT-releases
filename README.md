# Alpha Investing Tracker — Releases

This repository hosts the **installers** for Alpha Investing Tracker. The
app's built-in updater checks the latest release here and offers a one-click
update. The source code lives in a private repository; nothing personal or
private is ever published here.

## Install / update manually

Grab the newest release from the **Releases** page on the right:

- **Windows** — `AlphaInvestingTracker-Setup.exe` (double-click; SmartScreen
  may ask once: *More info → Run anyway*).
- **macOS** — `AlphaInvestingTracker-Mac.dmg` (open, drag AIT to
  Applications; first launch may need right-click → Open).

`manifest.json` carries the SHA-256 checksum of each installer — the in-app
updater verifies downloads against it before running anything.

Only the newest release is kept online.
